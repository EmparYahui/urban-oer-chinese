# 04 Assessment Protocols — Evaluating Learner Progress

## 🧠 Overview

This document defines the **operational assessment protocols** for evaluating learner outputs within the Urban Linguistic Landscape OER system.

It translates the assessment framework and rubrics into a **procedural evaluation workflow** applied to multimodal artefacts, mediation processes, and AR-mediated learning.

Assessment operates as a **continuous cycle of observation, interpretation, and evaluation**.

---

## 🧩 System role

This module functions as the **procedural layer of the assessment system**, connecting:

- task implementation (02 Learning Tasks)
- evidence generation (03 Evidence Mapping)
- evaluation criteria (02 Rubrics)
- research interpretation (03 Assessment system)

---

## 🔄 Assessment cycle

Assessment is structured as a four-stage recursive cycle:

---

### 1. Initial observation

Learners engage with urban linguistic input and generate first documentation.

**Focus of evaluation:**

- identification of relevant urban elements
- clarity of initial interpretation
- basic linguistic awareness
- relevance of selected input

---

### 2. Mediation and transformation

Learners reinterpret and transform meaning across languages and contexts.

**Focus of evaluation:**

- linguistic mediation quality
- cultural adaptation
- cross-linguistic reformulation
- audience orientation

---

### 3. Multimodal production

Learners construct multimodal artefacts combining text, image, and optional media.

**Focus of evaluation:**

- coherence across modes
- semiotic integration
- communicative effectiveness
- design of multimodal meaning

---

### 4. Reflection and evaluation

Final stage of assessment using rubrics and qualitative interpretation.

**Focus of evaluation:**

- rubric-based scoring (03 Rubrics)
- qualitative teacher/researcher notes
- OER potential assessment
- mediation effectiveness interpretation

---

## 🧠 Integration with Evidence Mapping

All assessment actions generate structured data in the form of:

👉 **Learning Evidence Units (LEUs)** (see 03 Evidence Mapping)

Each stage contributes to:

- contextual metadata (what/where/when)
- mediation processes (how meaning changes)
- multimodal outputs (final artefacts)
- evaluation encoding (rubric + qualitative data)

---

## 🧩 Assessment modalities

The system combines three complementary modes:

### 1. Formative assessment
- continuous feedback during task execution
- mediation guidance
- iterative improvement cycles

### 2. Summative assessment
- final evaluation of artefacts
- rubric-based scoring
- OER readiness evaluation

### 3. Research-informed assessment
- qualitative interpretation of mediation processes
- multimodal analysis
- AR spatial learning analysis

---

## 🧠 Best practices for implementation

- Provide feedback during mediation, not only at final output stage
- Combine rubric scoring with qualitative observation
- Encourage peer feedback as part of multimodal reflection
- Adapt criteria based on learner level (A1–A2 flexibility)
- Document assessment decisions as part of LEUs (03 Evidence Mapping)
