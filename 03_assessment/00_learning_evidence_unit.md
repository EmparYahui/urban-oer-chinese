# 00 Learning Evidence Unit — Conceptual Data Model

## 🧠 Overview

This document defines the **Learning Evidence Unit (LEU)** as the fundamental analytical entity of the Assessment module.

The LEU is the atomic unit for:

- learner activity
- multimodal artefacts
- mediation processes
- assessment decisions
- corpus construction

It connects learning design, assessment practice, and research analysis into a unified structure for educational data interpretation.

---

## 🧩 Definition

A **Learning Evidence Unit (LEU)** is a structured representation of a single instance of learner-generated evidence within a task-based, multimodal, and situated learning environment.

Each LEU corresponds to:

> one learner × one task moment × one multimodal output × one evaluative context

It is not a file or artefact, but a **structured analytical record**.

---

## 🧠 Structural model

Each LEU is composed of five interrelated dimensions:

---

### 1. Context

Defines the pedagogical and situational conditions in which the evidence is produced.

Includes:

- task identifier (linked to Module 02)
- learning sequence or phase
- learning environment (urban / classroom / hybrid)
- AR presence (if applicable)
- time of production

---

### 2. Input

Represents the stimulus or source material that triggers learner activity.

Includes:

- type of urban input (sign, image, text, audio, multimodal environment)
- description of the input
- languages present in the input
- cultural and spatial context of occurrence

---

### 3. Process (mediation layer)

Captures the cognitive, linguistic, and semiotic transformations performed by the learner.

Includes:

- interpretation strategies
- cross-linguistic mediation actions
- cultural adaptation processes
- selection and omission decisions
- multimodal design choices
- optional reflective notes produced during or after the task

This layer is central to research analysis.

---

### 4. Output

Represents the final learner-generated artefact.

Includes:

- modality (text / image / audio / video / AR / multimodal composition)
- content description
- languages used in production
- structural organisation of the artefact
- degree of multimodal integration

---

### 5. Evaluation

Links the LEU to assessment and interpretation.

Includes:

- rubric-based evaluation across four dimensions:
  - mediation quality
  - multimodal production
  - linguistic competence
  - OER potential
- qualitative assessment notes
- type of evaluator (teacher / researcher / peer)
- interpretive comments on learning evidence

---

## 🧠 Role in the system

The LEU is the **central analytical unit** of Module 03.

It connects:

- pedagogical design (Module 02)
- assessment instruments (rubrics + protocols)
- evidence mapping system
- corpus-based research analysis

---

## 🔄 System pipeline

The learning and research process is structured as:

> Task → Input → Mediation → Output → Evaluation → LEU → Corpus

This pipeline ensures traceability between:

- learning activity
- learner production
- assessment decisions
- research interpretation

---

## 📦 Function of the LEU

The LEU enables:

- structured documentation of learner production
- qualitative and comparative analysis of mediation
- multimodal corpus construction
- evaluation of OER quality and reusability
- longitudinal tracking of learner development

---

## 📌 Design principle

The LEU is a **conceptual analytical unit**, not a technical format.

It is designed to support:

- educational research
- qualitative corpus analysis
- multimodal discourse analysis
- assessment-informed pedagogy
