# Urban Linguistic Landscape OER

This Open Educational Resource (OER) supports beginner-level Chinese language learning (A1–A2) through situated, multimodal, and task-based pedagogy grounded in urban linguistic landscapes.

---

## 🧭 Overview

This project integrates:

- Urban linguistic landscapes as authentic learning input  
- Task-based language teaching (TBLT)  
- Multimodal meaning-making and hybrid production  
- Open Educational Resources (OER) principles  
- Optional augmented reality (AR) and geolocation components  

Learners engage in observation, mediation, and transformation of real-world linguistic environments into reusable educational artefacts.
