# Urban Linguistic Landscape OER

This Open Educational Resource (OER) supports research and teaching in:

- Second Language Acquisition  
- Multimodal Learning  
- Linguistic Landscapes  

## Citation

Liu Zhou, E. Y. (2026).  
Urban Linguistic Landscape OER for Task- and Project-Based Chinese Learning (A1–A2).  
Zenodo. https://doi.org/10.5281/zenodo.19535648  

## Description

This repository contains task designs, assessment rubrics, templates, and learner-generated artefacts developed within an urban, multimodal, task-based learning framework.
