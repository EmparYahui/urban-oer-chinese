# 📁 06 Examples

## 🧩 What this module is

This module presents a curated corpus of learner-generated artefacts produced during the implementation of the instructional model described throughout this repository.

Rather than serving as exemplary student work, these artefacts provide empirical evidence of learning processes, illustrate how the instructional design operates in practice, and demonstrate the types of multimodal products that learners can create through authentic, task-based activities.

Each example can be examined from three complementary perspectives:

- **Researchers** → as empirical evidence supporting the analysis of learner production, multimodal communication, and instructional design.
- **Teachers** → as implementation examples illustrating how learning tasks, assessment strategies, and digital tools can be adapted to different educational contexts.
- **Students** → as reference artefacts that clarify expectations, inspire multimodal production, and support self-assessment.

## 🚀 How to use

Choose the perspective that best matches your needs:

- 🔬 **Researchers** → analyse learner outputs as empirical evidence of language development, multimodal production, and situated learning.
- 👩‍🏫 **Teachers** → explore examples to understand how the instructional model can be implemented, adapted, and assessed in practice.
- 🎓 **Students** → examine previous artefacts to understand task expectations, production standards, and possible approaches to multimodal communication.
