# 07 Example Use Cases — Urban Linguistic Landscape OER

## 🧠 Overview

This document explores how **learner-generated artefacts** from the **Urban Linguistic Landscape OER** can be used in a variety of educational settings. It includes use cases for teachers, researchers, and instructional designers to implement artefacts into their own practice.

---

## 🧩 Use Cases

### 1. **Teacher Use Case**
- **Scenario**: A teacher is looking for example artefacts to show students the expected level of linguistic mediation and multimodal production.
- **Solution**: The teacher uses the **Urban Sign Mediation** artefact to illustrate how urban input can be transformed into a language learning resource. The teacher highlights how the artefact integrates text, image, and cultural interpretation.

### 2. **Researcher Use Case**
- **Scenario**: A researcher is analyzing the effectiveness of **AR integration** in urban learning.
- **Solution**: The researcher uses the **AR-Linked Geolocated Artefact** to study how geolocation and AR contribute to situated learning. They collect data on how students interact with the artefact in different urban environments.

### 3. **Instructional Designer Use Case**
- **Scenario**: An instructional designer is creating a new urban language learning task and needs an example artefact to guide the design process.
- **Solution**: The instructional designer reviews the **Multimodal Infographic** artefact to understand how to structure a task that integrates text and visuals for beginner Chinese learners. They adapt the example to suit their own learning objectives.

---

## 🧩 Purpose

This document demonstrates practical use cases for **learner-generated artefacts**, showing how they can be applied in real-world educational settings to enhance the learning experience.

---

## 🧩 Notes for Adaptation

- Examples can be adapted for use in a variety of educational contexts (e.g., online learning, face-to-face instruction).
- Use cases can be tailored to suit different proficiency levels and teaching objectives.
