# 🧩 AR Integration in Learning Design

## 🧠 Overview

This document explains how AR is integrated into the **learning sequence design**.

AR is embedded in task progression, not added externally.

---

## 🔄 Learning sequence integration

AR operates across three stages:

### 1. Urban observation
- learners identify linguistic signs in real environments
- AR is used to contextualise input

### 2. Mediation and transformation
- learners reinterpret meaning across languages
- AR supports contextual anchoring

### 3. Multimodal production
- learners create AR-linked artefacts
- outputs combine text, image, and spatial reference

---

## 🧠 Pedagogical function

AR supports:

- situated meaning-making
- cross-cultural mediation
- multimodal composition
- contextual learning
