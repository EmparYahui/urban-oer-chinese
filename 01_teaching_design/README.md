# Teaching Design — Urban Linguistic Landscape OER (Chinese A1–A2)

This folder contains the core pedagogical design of the Open Educational Resources (OER) module.

## Purpose

To provide a structured overview of the instructional design behind the urban task-based learning intervention.

The design integrates:
- Urban linguistic landscapes as learning environments
- Task-Based Language Teaching (TBLT)
- Multimodal production
- Open Educational Resources (OER) principles

- Module overview
- Session plans
- Task sequencing
- Pedagogical rationale
