# Learning Sequence Model — Urban OER System

## 🧠 Overview

This module defines the temporal and cognitive progression of learning across the system.

---

## 🔁 Learning phases

### Phase 1 — Urban observation
Learners interact with real linguistic landscapes and identify meaningful semiotic elements.

### Phase 2 — Interpretation and mediation
Learners transform observed input into meaning through linguistic and cultural mediation.

### Phase 3 — Multimodal production
Learners produce structured artefacts combining linguistic and visual elements.

### Phase 4 — Spatial extension (AR)
Artefacts are linked to physical spaces using augmented reality tools.

### Phase 5 — Evaluation
Learner outputs are assessed using linguistic, multimodal, and OER criteria.

### Phase 6 — Open dissemination
Selected outputs are published as Open Educational Resources.

---

## 📊 Progression logic

- A1 level → guided observation and structured mediation
- A2 level → autonomous production and expanded mediation

---

## 🔄 System property

The sequence is cyclical, not linear:

> Outputs from Phase 6 can re-enter Phase 1 as new input data.
