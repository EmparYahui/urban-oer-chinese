# Learning Tasks — Urban Linguistic Landscape (Chinese A1–A2)

This folder contains the task sequences designed for beginner-level Chinese learners (A1–A2 CEFR).

The tasks are based on urban linguistic landscapes and are structured according to Task-Based Language Teaching (TBLT) principles.

## Purpose

To guide learners through a progressive sequence of activities that move from:
- observation of real-world linguistic input
- to multimodal production
- to the creation of Open Educational Resources (OER)

## Task Phases

### Phase 1 — Urban Exploration
Learners identify and document linguistic landscape elements in urban environments.

### Phase 2 — Multimodal Production
Learners transform collected data into multimodal artefacts (text, image, optional audio/video).

### Phase 3 — AR Integration
Learners connect digital artefacts to physical spaces using augmented reality tools.

## Output

Each task sequence results in learner-generated artefacts designed for:
- educational reuse
- adaptation
- open dissemination under CC licensing
