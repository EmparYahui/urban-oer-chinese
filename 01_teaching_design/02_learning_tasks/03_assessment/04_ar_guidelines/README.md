# Augmented Reality (AR) Guidelines — Urban Linguistic Landscape OER

This folder provides guidelines for integrating Augmented Reality (AR) into the Urban Linguistic Landscape OER module for beginner Chinese (A1–A2).

---

## Purpose

To support the connection between:
- learner-generated digital artefacts
- physical urban spaces
- situated language learning experiences

AR is used as a mediational layer that enhances contextualisation and engagement.

---

## Role of AR in the learning design

AR is not treated as a standalone technology, but as a contextual mediator that:
- links multimodal artefacts to real-world locations
- supports situated meaning-making
- enhances learner engagement with urban environments

---

## Implementation overview

Learners:
1. Create multimodal artefacts (text, image, optional audio/video)
2. Assign each artefact to a physical location (geolocation)
3. Upload content to an AR platform
4. Access artefacts in situ via mobile devices

---

## Technical considerations

- Use accessible AR tools compatible with mobile devices
- Ensure geolocation accuracy at a basic functional level
- Prioritise usability over technical complexity

---

## Pedagogical note

The value of AR lies not in technological sophistication, but in its ability to recontextualise language learning within lived urban experience.

---

## Note

This framework is adaptable to different AR platforms and can be reused in other task-based language learning contexts.
