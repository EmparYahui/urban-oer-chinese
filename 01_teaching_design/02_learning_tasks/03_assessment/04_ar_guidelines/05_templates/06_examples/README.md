# Examples — Urban Linguistic Landscape OER

This folder contains anonymised examples of learner-generated artefacts created within the Urban Linguistic Landscape OER module (Chinese A1–A2).

---

## Purpose

To illustrate how learning tasks are translated into multimodal outputs and how these outputs can be transformed into reusable Open Educational Resources (OER).

---

## Types of examples included

- Urban linguistic landscape entries
- Multimodal artefacts (text + image + optional audio/video)
- AR-linked learning outputs
- Adapted versions for reuse in other contexts

---

## Pedagogical function

These examples serve to:
- demonstrate expected output formats
- support teacher adaptation of tasks
- illustrate levels of multimodal and linguistic development
- guide OER transformation processes

---

## Anonymisation

All learner data included in this section is fully anonymised in accordance with ethical research guidelines.

---

## Note

Examples are not model answers, but authentic productions illustrating variation in learner performance and design potential.
