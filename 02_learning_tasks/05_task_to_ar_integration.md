# 05 Task to AR Integration — Spatial Mediation Layer

## 🧠 Overview

This document defines how learning tasks are extended into augmented reality (AR) environments.

---

## 🌍 Integration logic

Tasks are extended into AR when:

- spatial context is relevant  
- meaning requires environmental anchoring  
- multimodal interpretation benefits from location  

---

## 🧭 AR integration modes

### 1. Annotation mode
Adding digital labels to physical urban elements.

### 2. Contextualisation mode
Linking meaning to specific spatial locations.

### 3. Interaction mode
Learners engage with AR layers to reinterpret input.

---

## 🔗 Task connection

- Observational tasks → AR annotation  
- Mediation tasks → AR contextualisation  
- Production tasks → AR embedding of artefacts  

---

## 🧠 System role

This module connects operational tasks with spatial mediation infrastructure.

## 📦 Output

- AR integration rules
