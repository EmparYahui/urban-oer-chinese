# 🌍 Start Here  
## Urban Linguistic Landscape OER (A1–A2 Chinese)

This resource introduces a complete open pedagogical system for task-based Chinese learning grounded in urban linguistic landscapes, multimodality, mediation, and augmented reality.

You do not need to read the theoretical folder first.

This page is the entry point.

---

## 🎯 What this OER is

This is a **complete learning system**, not a textbook or collection of activities.

It combines:

- real urban environments as learning input  
- task-based and project-based learning  
- multimodal language production  
- mediation between languages and cultures  
- augmented reality (optional layer)  
- open educational resource production  

Learners work with real-world language in the city and transform it into meaningful multimodal outputs.

---

## 🧭 What learners do

Across the learning cycle, students will:

1. Observe real urban language in context  
2. Collect linguistic and visual data  
3. Interpret meaning and cultural context  
4. Transform meaning across languages  
5. Produce multimodal artefacts (text + image + design)  
6. Share outputs as open resources  

---

## 🧩 System structure (simple view)

The system has six parts:

## 🔁 Learning cycle (core idea)

All learning follows a simple cycle:

1. Go into the city / real environment  
2. Identify linguistic elements  
3. Interpret and mediate meaning  
4. Create multimodal output  
5. Reflect and refine  
6. Share as open resource  

Then repeat.

---

## 🌐 What makes this system different

This OER is not:

- a fixed textbook  
- a list of exercises  
- a linear course  

It is:

- a **cyclical learning system**  
- a **city-based language environment**  
- a **multimodal production framework**  
- a **generator of open educational resources**  

---

## 📌 Before you begin

You will need:

- access to an urban environment (physical or virtual)

- willingness to interpret language in context  
- openness to multimodal production
