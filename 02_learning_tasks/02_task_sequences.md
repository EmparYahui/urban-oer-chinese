# 02 Task Sequences — Urban Linguistic Landscape OER

## 🧠 Overview

This document defines complete learning sequences that operationalise the pedagogical design into structured urban learning activities.

It can be used independently, without reading theoretical materials.

---

## 🚀 How to use this document

This document is ready to use in teaching and learning contexts.

Start with Sequence 1.

- Teachers → select and implement sequences directly
- Learners → follow tasks step by step in urban or virtual environments
- Designers → adapt sequences to different languages or contexts

---

## 🔁 Sequence structure

Each sequence follows a cyclical progression:

1. Urban input selection  
2. Observation and documentation  
3. Linguistic and cultural mediation  
4. Multimodal production  
5. Optional AR contextualisation  
6. Reflection and reuse as OER  

---

## 🌍 Sequence 1 — Urban reading

### Focus
Interpreting linguistic landscapes as semiotic systems.

### Task flow

1. Identify a linguistic element in the urban environment  
2. Photograph or capture the element  
3. Describe its form and function  
4. Interpret its meaning in context  
5. Reflect on cultural or communicative implications  

### Output
- Annotated image  
- Short interpretive note (L1 or supported L2)

---

## 🌍 Sequence 2 — Mediation cycle

### Focus
Transforming meaning across languages and contexts.

### Task flow

1. Select one urban linguistic item  
2. Identify meaning and communicative intent  
3. Reformulate it in simplified Chinese (A1–A2 level)  
4. Add brief mediation notes (optional L1 support)

### Output
- Short Chinese text  
- Optional bilingual notes

---

## 🌍 Sequence 3 — Multimodal construction

### Focus
Creating multimodal artefacts from mediated meaning.

### Task flow

1. Use mediated Chinese text  
2. Combine with image or visual representation  
3. Design a structured multimodal artefact  

### Output
- Digital poster / infographic / annotated image  

---

## 🌍 Sequence 4 — AR-enhanced interaction

### Focus
Spatialising meaning through augmented reality.

### Task flow

1. Select a multimodal artefact  
2. Link it to a physical urban location (or simulated map)  
3. Add contextual meaning through AR annotation  
4. Reflect on spatial meaning-making  

### Output
- AR-linked artefact or geolocated resource  

---

## 🔁 Reflection phase (applies to all sequences)

After each sequence:

- What did you observe?  
- How did meaning change through mediation?  
- How does space influence interpretation?  
- How could this be reused as an educational resource?  

---

## ♻️ Output use (OER logic)

All outputs can be reused as:

- learning evidence  
- teaching materials  
- multimodal artefacts  
- research data on language learning  

Outputs are designed for adaptation across languages, cities, and proficiency levels.
