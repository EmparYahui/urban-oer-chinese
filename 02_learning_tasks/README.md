# 🌍 02 Learning Tasks

## 🧩 What this module is

This module operationalises the pedagogical design into **concrete task sequences**.

Tasks are situated in urban linguistic landscapes and structured for mediation, multimodal production, and open educational reuse.

## 🧠 Pedagogical principle

This module transforms urban space into a structured learning environment where:

- tasks structure action  
- mediation structures meaning  
- multimodality structures output  
- AR extends spatial learning  
- OER defines reuse logic  

---

## 🔁 Learning logic

Urban input → Task → Mediation → Multimodal output → (optional AR) → Open reuse
