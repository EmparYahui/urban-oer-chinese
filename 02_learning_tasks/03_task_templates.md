# 03 Task Templates — Urban OER System

## 🧠 Overview

This document provides reusable templates for designing tasks within the urban linguistic landscape framework.

---

## 🧩 Standard task structure

Each task includes:

- Urban input  
- Learning objective  
- Mediation process  
- Expected output  
- Multimodal format  
- OER potential  

---

## 🧪 Template 1 — Observation task

- Input: urban sign / text / image  
- Action: identify and describe elements  
- Output: annotated description  

---

## 🧪 Template 2 — Mediation task

- Input: source text (L1 or L2)  
- Action: transform meaning into target language  
- Output: mediated version  

---

## 🧪 Template 3 — Multimodal production task

- Input: mediated content  
- Action: combine text + visual representation  
- Output: multimodal artefact  

---

## 🧠 System role

This module ensures replicability and adaptation across contexts and languages.
