# 03 Assessment — Urban Linguistic Landscape OER

## 🧠 Overview

This module defines the assessment framework for all learning tasks within the Urban Linguistic Landscape OER system.

It evaluates both **language learning outcomes** and **process-based multimodal mediation**, including AR-enhanced production.

Assessment is fully integrated into pedagogy and aligned with open educational principles.

---

## 🚀 How to use this module

This assessment framework can be used independently.

- Teachers → apply rubrics directly to student outputs  
- Learners → self-assess progression across sequences  
- Researchers → analyse mediation, multimodality, and task performance  

No theoretical reading is required.

---

## 🎯 Assessment principles

All evaluation is based on the following principles:

- **Process-oriented assessment** (not only final product)  
- **Mediation-focused evaluation** (how meaning is transformed)  
- **Multimodal competence** (integration of modes)  
- **Contextual validity** (urban + situated learning)  
- **OER reusability** (clarity and adaptability of outputs)  
- **A1–A2 CEFR alignment** (beginner level progression)

---

## 🧩 Assessment dimensions

Each learner output is evaluated across five dimensions:

### 1. Linguistic development
- Accuracy at A1–A2 level  
- Vocabulary appropriateness  
- Basic syntactic control  

---

### 2. Mediation quality
- Ability to transform meaning across languages  
- Clarity of interpretation  
- Cultural appropriateness of reformulation  

---

### 3. Multimodal coherence
- Integration of text and image  
- Clarity of visual organisation  
- Consistency between modes  

---

### 4. Urban contextual engagement
- Relevance to real linguistic landscape  
- Quality of observation and interpretation  
- Awareness of spatial meaning  

---

### 5. OER potential
- Reusability of output  
- Clarity for external users  
- Adaptability to other contexts or languages  

---

## 🧪 Assessment instruments

### Instrument 1 — Analytic rubric

Each dimension is rated:

- 1 = emerging  
- 2 = developing  
- 3 = competent  
- 4 = strong  

---

### Instrument 2 — Mediation checklist

Used specifically for Tasks 2–4:

- Meaning preserved across transformation  
- Cultural context considered  
- Simplification appropriate for A1–A2  
- Communication remains clear  

---

### Instrument 3 — Multimodal evaluation grid

Used for production tasks:

- Visual clarity  
- Text-image alignment  
- Structural readability  
- Communicative effectiveness  

---

## 🔁 Formative assessment cycle

Assessment is continuous and iterative:

1. Task completion  
2. Feedback on mediation and meaning  
3. Revision of artefact  
4. Re-submission or reuse in next sequence  

---

## 🌍 AR-specific assessment criteria

For AR-linked tasks:

- Spatial relevance of annotation  
- Clarity of location-based meaning  
- Integration between physical and digital layers  
- Enhancement of interpretation through spatial context  

---

## ♻️ OER evaluation criteria

Outputs are considered valid OER when they meet:

- clarity for external users  
- linguistic transparency  
- multimodal readability  
- contextual independence  
- adaptability to other teaching contexts  

---

## 🧭 Assessment workflow

Urban input → Task execution → Mediation → Multimodal output → AR (optional) → Evaluation → Revision → OER classification

---

## 🧠 Role of assessment in the system

Assessment is not external.

It functions as:

- a learning mechanism  
- a feedback system  
- a quality filter for OER production  
- a research instrument for mediation analysis
