# 04 Urban Task Types — OER Operational Taxonomy

## 🧠 Overview

This document defines the operational taxonomy of task types used in the Urban Linguistic Landscape OER system.

It provides a structured classification of all learning actions across observation, mediation, production, and OER creation.

---

## 🧩 Core task taxonomy

### 1. Observational tasks
Focus: identifying and documenting urban linguistic input.

- urban signs
- public texts
- visual-linguistic artefacts
- spatial language data

**Output examples:**
- annotated images
- field notes
- descriptive logs

---

### 2. Interpretive tasks
Focus: constructing meaning from urban input.

- linguistic analysis of signs
- cultural interpretation
- contextual inference
- meaning negotiation

**Output examples:**
- interpretation notes
- short analytical texts
- bilingual explanations

---

### 3. Mediation tasks
Focus: transforming meaning across languages and perspectives.

- L1 ↔ L2 reformulation
- simplification / expansion
- intercultural adaptation
- audience adjustment

**Output examples:**
- mediated texts in Chinese (A1–A2)
- bilingual versions
- reformulated explanations

---

### 4. Production tasks
Focus: multimodal creation of educational artefacts.

- text + image composition
- infographic design
- multimodal storytelling
- structured knowledge artefacts

**Output examples:**
- posters
- digital cards
- annotated visual artefacts

---

### 5. AR-linked tasks
Focus: spatial embedding of learning.

- geolocated interpretation
- AR annotation of urban elements
- spatial mapping of meaning
- context-aware artefact placement

**Output examples:**
- AR tags
- location-based artefacts
- spatial learning records

---

### 6. OER publication tasks
Focus: preparing outputs for reuse and dissemination.

- anonymisation
- formatting for reuse
- licensing preparation
- pedagogical documentation

**Output examples:**
- reusable OER packages
- shared learning artefacts
- repository-ready materials

---

## 🔁 System role

This module defines the **operational vocabulary of action** for the entire system.

It translates theory (Layer 0–1) into executable pedagogical units.
