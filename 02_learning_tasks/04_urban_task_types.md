# 04 Urban Task Types — OER Operational Taxonomy

## 🧠 Overview

This document defines the taxonomy of task types used in urban linguistic landscape learning.

---

## 🧩 Task categories

### 1. Observational tasks
Identify and document linguistic elements in urban environments.

### 2. Interpretive tasks
Analyse meaning and cultural context of urban signs and texts.

### 3. Mediation tasks
Transform meaning across languages, registers, or communicative purposes.

### 4. Production tasks
Create multimodal artefacts combining text, image, or other media.

### 5. AR-linked tasks
Connect artefacts to physical spaces using augmented reality tools.

### 6. OER publication tasks
Prepare learner outputs for reuse and open dissemination.
