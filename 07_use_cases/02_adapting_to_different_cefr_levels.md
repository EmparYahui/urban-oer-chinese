# Adapting the Model to Different CEFR Levels

## Purpose

This guide explains how the instructional model presented in this repository can be adapted to different levels of language proficiency while preserving its pedagogical foundations. Although the original implementation was designed for beginner learners, the model is proficiency-independent. Its core principles remain stable across levels, whereas linguistic complexity, task demands, learner autonomy, and assessment criteria should be progressively adjusted.

The objective is not to simplify or expand the instructional sequence indiscriminately, but to align communicative expectations, learning outcomes, and multimodal production with learners' evolving linguistic competence.

---

# Core Pedagogical Principles That Remain Constant

Regardless of proficiency level, the following principles should remain unchanged:

* Learning is situated in authentic or meaningful environments.
* Language develops through purposeful communicative tasks.
* Learners construct knowledge through multimodal production.
* Collaboration and peer interaction support learning.
* Assessment is continuous, formative, and integrated into the learning process.
* Learner-produced artefacts can become reusable Open Educational Resources whenever appropriate.

These principles define the instructional architecture of the model and should be preserved across all proficiency levels.

---

# Adapting the Learning Outcomes

Learning outcomes should evolve from basic language production towards increasingly sophisticated communicative competence.

| CEFR Level | Typical learning outcomes                                                                                                                 |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| A1         | Identify, name, describe, and produce simple formulaic expressions in authentic contexts.                                                 |
| A2         | Describe everyday situations, provide basic explanations, and interact using familiar language.                                           |
| B1         | Narrate experiences, justify opinions, and interpret cultural phenomena in context.                                                       |
| B2         | Analyse authentic materials, compare perspectives, and communicate with increasing fluency and precision.                                 |
| C1         | Produce extended multimodal texts, critically evaluate linguistic and cultural phenomena, and adapt communication to different audiences. |
| C2         | Design, curate, and publish complex multilingual artefacts demonstrating advanced communicative, intercultural, and digital competence.   |

---

# Adapting the Tasks

Task complexity should increase progressively while maintaining authentic communicative purposes.

| CEFR Level | Examples of tasks                                                                                                                                   |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| A1         | Label photographs, identify places, write captions, complete guided observations.                                                                   |
| A2         | Produce short descriptions, simple interviews, digital maps, and basic multimedia presentations.                                                    |
| B1         | Create thematic urban guides, podcasts, subtitled videos, and comparative reports.                                                                  |
| B2         | Conduct field research, analyse linguistic landscapes, produce documentaries, and develop collaborative digital projects.                           |
| C1–C2      | Design public-facing OERs, conduct independent investigations, curate multilingual exhibitions, and produce research-informed multimodal resources. |

---

# Adapting the Linguistic Demands

Increasing proficiency should be reflected through greater linguistic complexity rather than through changes in the pedagogical model itself.

Teachers may progressively adjust:

* lexical range;
* grammatical complexity;
* discourse organisation;
* intercultural interpretation;
* critical reflection;
* communicative autonomy;
* audience awareness.

Authenticity of communication should remain the primary objective at every level.

---

# Adapting the Assessment

Assessment criteria should evolve alongside learners' proficiency while maintaining the same assessment philosophy.

| Lower levels (A1–A2)                  | Higher levels (B1–C2)                                 |
| ------------------------------------- | ----------------------------------------------------- |
| Accuracy of basic language production | Precision, fluency, and communicative effectiveness   |
| Completion of guided tasks            | Independent problem solving                           |
| Basic multimodal integration          | Sophisticated multimodal orchestration                |
| Participation and collaboration       | Leadership, collaboration, and project management     |
| Guided reflection                     | Critical self-evaluation and metacognitive reflection |

Rubrics should therefore be recalibrated rather than replaced when moving across proficiency levels.

---

# Suggested Progression

As learners become more proficient, progression should occur simultaneously across several dimensions.

| Dimension     | A1              | A2                     | B1                     | B2                  | C1–C2                           |
| ------------- | --------------- | ---------------------- | ---------------------- | ------------------- | ------------------------------- |
| Language      | Formulaic       | Everyday communication | Narrative              | Analytical          | Critical and specialised        |
| Tasks         | Guided          | Structured             | Semi-independent       | Independent         | Self-directed                   |
| Collaboration | Teacher-guided  | Shared responsibility  | Collaborative planning | Autonomous teamwork | Project leadership              |
| Multimodality | Basic           | Integrated             | Expanded               | Sophisticated       | Professional-quality production |
| Assessment    | Guided feedback | Formative assessment   | Reflective revision    | Self-regulation     | Critical evaluation             |

---

# Implementation Checklist

Before adapting the model, verify that:

* ☐ communicative objectives match learners' proficiency;
* ☐ task complexity increases without changing the pedagogical principles;
* ☐ multimodal production remains central;
* ☐ learner autonomy is appropriate for the target level;
* ☐ assessment criteria are aligned with expected linguistic performance;
* ☐ authentic communication remains the primary learning objective.

---

# Key Takeaway

The instructional model is not tied to a specific proficiency level. Its transferability depends on preserving the pedagogical architecture while progressively adjusting linguistic demands, task complexity, learner autonomy, and assessment expectations. Teachers should therefore adapt the level of challenge—not the underlying instructional design—to ensure meaningful learning across the entire CEFR continuum.
