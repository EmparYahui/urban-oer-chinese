# 05 OER Output Template — Urban Linguistic Landscape OER

## Overview

This template helps educators create **Open Educational Resources (OER)** that are pedagogically clear, reusable, and adaptable. It guides the design of artefacts that can be shared and used across contexts.

---

## Template Structure

| **Component**             | **Description**                                                                                      |
|---------------------------|------------------------------------------------------------------------------------------------------|
| **Artefact title**         | Provide a concise and descriptive title for the OER.                                                 |
| **Core learning objective**| Specify the key learning objectives associated with the artefact.                                    |
| **Content description**    | Provide a brief summary of the content in the artefact.                                               |
| **Multimodal components**  | List the modes used (e.g., text, images, audio).                                                    |
| **Adaptability**           | Describe how the artefact can be adapted or reused in different educational contexts.                |
| **Pedagogical considerations**| Outline any specific pedagogical goals or strategies addressed by the artefact.                     |

---

## Purpose

This template ensures that artefacts are designed to be **reusable** and **adaptable**, supporting broader educational goals and practices.

---

## Notes for Adaptation

- The artefact can be expanded or simplified depending on the context.
- Can be used with any subject area or language focus.
