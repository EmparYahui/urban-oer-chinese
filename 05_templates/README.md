# Templates — Urban Linguistic Landscape OER

This folder contains reusable templates for the creation of multimodal learning artefacts within the Urban Linguistic Landscape OER module (Chinese A1–A2).

---

## Purpose

To support teachers and learners in:
- structuring multimodal outputs
- ensuring consistency across artefacts
- facilitating reuse and adaptation in different educational contexts

---

## Template types

### 1. Multimodal artefact template
A structured format for combining:
- text (description and interpretation)
- image (urban linguistic input)
- optional audio/video
- contextual metadata (location, date, interpretation notes)

---

### 2. Linguistic landscape entry template
Used during Phase 1 (urban exploration) to document:
- signage or visual text
- location
- linguistic interpretation
- cultural notes

---

### 3. OER publication template
Supports conversion of learner outputs into reusable educational resources:
- pedagogical description
- learning objectives
- reuse potential
- licensing information

---

## Pedagogical note

Templates are designed to reduce cognitive load while ensuring pedagogical consistency and enhancing the reusability of learner-generated artefacts.

---

## Note

These templates are adaptable and can be modified for other languages, contexts, or task-based learning designs.
