# 02 Task Design Template — Urban Linguistic Landscape OER

## 🧠 Overview

This template helps educators design tasks based on real-world urban environments, aligned with **Task-Based Language Teaching (TBLT)** principles. The goal is to integrate **linguistic mediation**, **multimodal production**, and **AR technologies** into urban exploration tasks.

---

## 🧩 Template Structure

| **Component**               | **Description**                                                                                     |
|-----------------------------|-----------------------------------------------------------------------------------------------------|
| **Task title**              | Provide a concise and descriptive title for the task.                                               |
| **Urban context**            | Specify the urban environment (e.g., streets, parks, advertisements, street signs).                 |
| **Linguistic focus**         | Define the linguistic elements targeted by the task (e.g., grammar, vocabulary).                    |
| **Target CEFR level**        | Specify the CEFR level (e.g., A1, A2).                                                             |
| **Input type**               | Describe materials (e.g., signs, advertisements).                                                  |
| **Student action**           | What will the students do during the task (e.g., observe, translate, interpret)?                    |
| **Expected output**          | Describe the expected output (e.g., written reports, oral presentations).                          |
| **Mediation focus**          | Key aspects of linguistic mediation (e.g., simplification, reformulation, adaptation).              |
| **Multimodal elements**      | List the modes involved (e.g., text, images, video).                                                |
| **Optional AR integration**  | Describe how AR will enhance the task (e.g., linking real-world locations to digital content).      |

---

## 🧩 Purpose

This template helps in structuring urban exploration tasks for meaningful language learning experiences, ensuring alignment with OER pedagogical goals.

---

## 🧩 Notes for Adaptation

- Adaptable to various urban environments.
- Can be used without AR integration.
- Flexible for different CEFR levels.
