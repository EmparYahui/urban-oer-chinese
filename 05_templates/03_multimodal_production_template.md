# 03 Multimodal Production Template — Urban Linguistic Landscape OER

## 🧠 Overview

This template provides a framework for designing multimodal learning artefacts, integrating **text, images, audio**, and optionally **AR technology**. The goal is to support students in creating coherent and meaningful multimodal artefacts that reflect their learning in real-world urban environments.

---

## 🧩 Template Structure

| **Component**               | **Description**                                                                                         |
|-----------------------------|---------------------------------------------------------------------------------------------------------|
| **Artefact title**           | Provide a concise and descriptive title for the artefact.                                                |
| **Core message**             | State the central idea or message to be conveyed through the artefact.                                  |
| **Text component**           | Include the linguistic content (e.g., written text in Chinese).                                         |
| **Visual component**         | Describe the visual elements (e.g., images, diagrams, graphics).                                        |
| **Audio/video**              | If applicable, provide a description of the audio or video content (e.g., narration, interviews).        |
| **Design rationale**         | Explain the pedagogical and design choices made in creating the artefact.                               |
| **Intended audience**        | Define the target audience for the artefact (e.g., peers, teachers, public).                            |
| **Reuse potential**          | Indicate how the artefact can be reused or adapted in other contexts.                                   |

---

## 🧩 Purpose

This template helps structure multimodal artefacts that integrate linguistic mediation, multimodal production, and cultural adaptation.

---

## 🧩 Notes for Adaptation

- Adaptable to various CEFR levels.
- Can be used without AR integration.
- Flexible for different urban environments and cultural contexts.
