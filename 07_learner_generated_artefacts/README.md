# Learner-Generated Artefacts

This section presents representative artefacts created by learners during the Urban Linguistic Landscapes project.

The artefacts were selected from the corpus analysed in the accompanying study and illustrate how mediation, multimodal composition, audience awareness, intercultural reflection, and OER reuse potential emerged throughout the learning process.

The examples are organised into four categories:

1. Restaurant Role-Play
2. Multimodal Urban Trail
3. Collaborative Mediation
4. Reflective Learning Evidence

Together, these artefacts document both the learning outcomes of the project and their potential value as reusable educational resources.
