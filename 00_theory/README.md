# Theory Layer Navigation Map
## Urban Linguistic Landscape OER (A1–A2)

This folder (`00_theory/`) contains the conceptual foundation of the entire OER system.

It is not a set of independent texts, but a **relational knowledge architecture** that defines how all pedagogical components interact.

---

## 🧭 1. Purpose of this layer

The theory layer defines:

- the epistemological assumptions of the project  
- the pedagogical logic of task design  
- the mediational structure of learning  
- the conditions for multimodal and AR-based learning  
- the principles of openness and reuse (OER)  
- the integrated system model

## 🔗 3. Dependency structure (how the system is built)

The theoretical system is hierarchical but also recursive.

### 🧩 Foundational level
- 00_core_framework → defines epistemological conditions

### 🌍 Contextual level
- 01_urban_linguistic_landscapes → defines learning environment
- 02_task_and_project_based_learning → defines pedagogical action

### 🔄 Mediational level
- 03_multimodality_and_mediation → defines semiotic transformation
- 04_ar_as_mediational_layer → extends mediation into space

### 📦 Output level
- 05_oer_open_pedagogy → defines reuse and openness

### 🧠 Integrative level
- 06_integrated_model → consolidates the system
