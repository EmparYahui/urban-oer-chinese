# 06 Integrated Model  
## Urban Linguistic Landscape OER System (A1–A2 Chinese)

This document synthesises the theoretical and pedagogical architecture of the Urban Linguistic Landscape OER system into a unified operational model.

It integrates the five foundational components defined in this folder:

- Urban Linguistic Landscapes  
- Task and Project-Based Learning  
- Multimodality and Mediation  
- AR as Mediational Layer  
- Open Educational Resources and Open Pedagogy  

---

## 🧩 1. System Overview

The model defines a **cyclical, multimodal, and spatially situated learning system** in which urban environments function as the primary semiotic infrastructure for language learning.

Learning is not understood as content acquisition, but as:

- situated interpretation of urban signs  
- mediated linguistic transformation  
- multimodal knowledge production  
- open dissemination of learner artefacts  

---

## 🧠 2. Core System Logic

The system is structured around four recursive operations:

### 1. Perception (Urban Input)
Learners engage with linguistic landscapes in real-world environments.

→ street signs  
→ public texts  
→ commercial signage  
→ institutional language  

---

### 2. Mediation (Interpretation Layer)
Learners transform observed input through linguistic and intercultural mediation.

→ translation  
→ reformulation  
→ meaning negotiation  
→ contextual interpretation  

---

### 3. Production (Multimodal Output)
Learners generate artefacts combining multiple semiotic modes:

→ text  
→ image  
→ spatial annotation  
→ optional audio/video  

---

### 4. Circulation (Open Release)
Learner outputs are transformed into reusable educational resources.

→ Open Educational Resources (OER)  
→ shared repositories  
→ public knowledge artefacts  

---
