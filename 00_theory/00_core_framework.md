# Core Theoretical Framework

## 1. System definition

This repository is a research-informed Open Educational Resources (OER) system designed for task-based Chinese language learning in urban contexts (A1–A2 CEFR level).

It integrates pedagogical design, urban linguistic landscapes, multimodal meaning-making, and augmented reality (AR) into a unified and layered instructional architecture.

The system operates as a structured pedagogical-research environment in which learning design, learner activity, and research evidence are co-constructed.

---

## 2. Epistemological stance

The system is based on the following assumptions:

- Language learning is situated in real-world semiotic environments
- Meaning emerges through social, spatial, and technological mediation
- Learning is task-driven, not content-transmission based
- Knowledge is constructed through interaction with multimodal urban input
- Learning processes are inherently distributed across people, space, and media

Knowledge is therefore not transmitted but co-constructed through engagement with urban environments and mediated artefacts.

---

## 3. Core theoretical foundations

This system is grounded in the integration of:

- Task-Based Language Teaching (TBLT)
- Urban Linguistic Landscape (ULL) research
- Multimodal and embodied learning theories
- Linguistic mediation in second language acquisition
- Open Educational Practices (OEP) and OER design principles
- Technology-enhanced and situated learning

These frameworks converge into a unified pedagogical architecture rather than operating as separate theoretical domains.

---

## 4. System architecture logic

The repository is structured as a layered system:

- Layer 0: Theoretical foundation
- Layer 1: Pedagogical design
- Layer 2: Learning tasks and implementation
- Layer 3: Assessment and evaluation
- Layer 4: Augmented reality mediation
- Layer 5: Reusable templates
- Layer 6: Empirical examples

Each layer is dependent on and derived from the previous one, forming a coherent vertical system of design, action, and evidence.

---

## 5. Role of urban linguistic landscapes

Urban linguistic landscapes function as:

- authentic semiotic environments
- sources of multimodal input
- sites of interpretation and mediation
- spaces for task execution and data generation

The city is treated as an active pedagogical system rather than a passive context.

---

## 6. Multimodality and mediation

Learning is understood as a multimodal process involving:

- linguistic mediation between languages
- transformation of perception into structured meaning
- integration of text, image, spatial awareness, and digital media

Meaning is distributed across modes and emerges through their interaction.

---

## 7. Augmented reality as mediational layer

Augmented reality operates as a bridge between:

- physical urban space
- digital representation
- learner-generated artefacts

It extends learning into situated and spatially anchored experiences, enabling contextualised meaning-making.

---

## 8. OER principle and openness

All learner-generated artefacts are considered potential Open Educational Resources.

The system is structured around openness as a design principle:

- reuse
- adaptation
- redistribution
- transformation

Openness is embedded in the architecture, not added as a final stage.

---

## 9. Pedagogical principle

The system is grounded in task-based learning:

- tasks structure learning processes
- learners engage with real-world input
- outputs are constructed through mediated production
- assessment evaluates both process and artefact quality

Tasks are therefore both instructional units and research instruments.

---

## 10. Integrative system logic

All components operate as a unified cyclical system:

- design informs implementation
- implementation generates artefacts
- artefacts become evidence
- evidence informs redesign

The system is iterative, adaptive, and continuously evolving.

---

## 11. Core identity of the system

This repository is not:

- a static teaching resource collection
- a linear course package
- a set of independent modules

It is:

- a cyclical pedagogical system
- a research-informed educational infrastructure
- a living Open Educational Resources ecosystem

---

## 12. Final conceptual statement

Urban space, task-based learning, multimodality, mediation, and augmented reality are not separate components.

They form a single integrated system in which:

- the city becomes a learning environment
- tasks become design mechanisms
- learner outputs become reusable knowledge artefacts
- openness becomes the structural logic of the entire system
