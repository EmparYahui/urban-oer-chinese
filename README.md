# Urban Linguistic Landscape OER for Task-Based Chinese Learning (A1–A2)

This repository presents a research-informed Open Educational Resources (OER) system for beginner-level Chinese (A1–A2 CEFR). It is based on an empirical study exploring how urban linguistic landscapes, multimodal production, mediation, and augmented reality (AR) can be integrated into task-based language learning to support the co-creation of reusable educational resources.

---

## 🧩 System overview

This repository constitutes a research-informed pedagogical system in which Open Educational Resources (OER) are not treated as outputs, but as a **design principle embedded throughout the learning process**.

It integrates three interdependent layers:

- **Pedagogical design layer** → instructional planning and task architecture  
- **Implementation layer** → urban task sequences and learning activities  
- **Evidence layer** → assessment instruments and learner-generated artefacts  

These layers operationalise a unified model in which design, mediation, production, and evaluation are structurally interconnected.

---

## 📌 Research context

This OER is derived from a mixed-methods case study examining how task-based learning in urban environments can support the co-creation of reusable educational resources.

The study is situated at the intersection of:

- Urban Linguistic Landscapes (ULL)  
- Task-Based Language Teaching (TBLT)  
- Linguistic and intercultural mediation  
- Multimodal production and hybrid writing  
- Open Educational Resources (OER) and open pedagogy  
- Augmented Reality (AR) as a mediational infrastructure  

---

## 🧠 Conceptual contribution

The project conceptualises OER as a **design principle embedded across instructional planning, implementation, and assessment**.

Within this model:

- tasks are designed as potentially reusable artefacts from the outset  
- learner outputs are treated as candidates for educational reuse  
- openness structures pedagogical design and evaluation  
- urban environments function as semiotic and pedagogical spaces  
- AR operates as a contextual mediational layer linking physical and digital spaces

## 🔄 Pedagogical workflow

The system follows a cyclical model of situated production and reuse:

1. Urban input → engagement with linguistic landscapes  
2. Mediation → interpretation and meaning negotiation  
3. Multimodal production → creation of structured artefacts  
4. AR contextualisation → spatial and digital linkage of meaning  
5. Evaluation → linguistic and OER quality assessment  
6. Open publication → release of artefacts as OER

- modifying AR tools and platforms
